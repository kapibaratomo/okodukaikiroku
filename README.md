# お小遣い記録サイト

PASMOと現金を分けて管理できる、シンプルでミニマルなお小遣い記録のWebアプリケーションです。
ブラウザの `localStorage` を使用してデータを保存するため、サーバーやデータベースは不要です。

## 特徴
- 💳 **PASMO / 現金 の一元管理**: どちらで使った・貰ったかを分けて記録可能
- 📊 **自動計算**: 総残高だけでなく、PASMOと現金それぞれの残高を自動計算
- ☁️ **完全クラウド同期**: データはすべて指定のGitHubリポジトリ（API経由）に安全に保存されます
- 🔄 **どこからでもアクセス**: GitHub Tokenを設定するだけで、スマホや他のPCから常に最新の履歴が見られます
- 🚀 **GitHub Pages対応**: このリポジトリをPushするだけで、自動でWebサイトとして公開されます

## 使い方

### 1. サイトにアクセスする
GitHub Pages（後述の設定後）のURLにアクセスするか、ローカルの `index.html` をブラウザで開いてください。

### 2. 記録をつける
「PASMO / 現金」および「収入 / 支出」を選び、金額とメモを入力して「追加する」ボタンを押します。
最新の記録が履歴の上部に追加されます。

### 3. 他のPCやスマホとデータを共有する（自動同期）
このサイトはデータを「GitHubの裏側（リポジトリのJSONファイル）」に自動保存します。
これを有効にするには **GitHub Personal Access Token (PAT)** が必要です。

#### Tokenの取得・設定方法
1. GitHubの [Developer Settings](https://github.com/settings/tokens/new) にアクセス
2. **Note** に `okodukai-sync` 等を入力
3. **Expiration** を `No expiration` （無期限）などに設定
4. **Select scopes** で `repo` （一番上のチェックボックス）にチェックを入れて一番下の「Generate token」を押す
5. 表示された `ghp_...` から始まるTokenをコピー
6. お小遣いサイト右上の **⚙️（歯車マーク）** を押し、Tokenを貼り付けて「設定を保存して同期」を押す

一度設定すれば、ブラウザに保存され、次からは自動で最新のデータが同期されます！✨

## GitHub Pagesでの公開方法（デプロイ）

このフォルダをGitHubにアップロード（Push）するだけで、設定した自動化スクリプト（`.github/workflows/deploy.yml`）が自動でサイトを公開してくれます。

### 初回設定
1. GitHub リポジトリの **Settings (設定)** タブを開く
2. 左のメニューから **Pages** を選択
3. **Source** を `GitHub Actions` に変更
   *(自動デプロイが有効になります)*
4. **変更を反映（デプロイ）させる手順:**
   今後、ローカルのパソコンでソースコードを変更した場合は、ターミナル（コマンドプロンプトやPowerShell）でこのフォルダ（`okodukai`）を開き、以下のコマンドを順番に実行することで、自動的にWebサイトに新しいコードが反映されます。

   ```bash
   # 1. 変更したすべてのファイルを追加
   git add .
   
   # 2. 変更内容に名前（コミットメッセージ）をつける
   git commit -m "変更内容の説明（例：デザインを修正）"
   
   # 3. GitHubへコードをPush（アップロード）する
   git push origin main
   ```
   Pushをしてから1〜2分待つと、GitHub Actionsが自動でデプロイ（Webサイトの更新）を行ってくれます！
